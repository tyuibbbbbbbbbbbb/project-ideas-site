# אתר הפרויקטים 💡

אתר לשיתוף רעיונות לפרויקטים — מי שיש לו רעיון אבל לא יכול לפתח אותו מעלה אותו לאתר, ומי שמחפש פרויקט לעבוד עליו יכול לבחור מהרשימה.

## תכונות
- העלאת רעיון חדש (כותרת, תיאור, שם)
- לייק 👍 ודיסלייק 👎 לכל רעיון
- הרעיונות ממוינים לפי ניקוד (לייקים פחות דיסלייקים) — הפופולרי ביותר למעלה
- רעיון עם מעל 10 דיסלייקים עובר אוטומטית ל"בדיקה" ולא ניתן להצביע עליו יותר

## הרצה
```bash
npm install
npm start
```
האתר יעלה על http://localhost:3000

הנתונים נשמרים בקובץ `data.json`.
